Data Structures and Algorithms

Solving an updated version of the traveling salesman problem.

Project Overview

As part of my coursework in Data Structures and Algorithms, I undertook a modern interpretation of the classic traveling salesman problem. The project tasked me with devising an efficient solution for delivering packages to multiple addresses using multiple drivers and vehicles.

Harnessing Python, I constructed a bespoke hash table to manage package data from the ground up. Employing the nearest neighbor algorithm, I crafted optimized delivery routes that fulfilled all assignment specifications. Beyond the coding, I was required to evaluate each script component for its time and space complexity, showcasing my competency in coding efficiency. Additionally, I composed a comprehensive write-up detailing the rationale behind my decisions, demonstrating both technical acumen and strategic thinking.

This project not only honed my skills in data structures and algorithms but also underscored my ability to tackle complex real-world challenges with innovative solutions.

Scheduling Application

A GUI-based desktop application to manage appointments.

Project Overview

For a project in GUI-based application development, I created an application tailored to the specific needs of a fictional company. Coded in Java and SQL, the application incorporated various functionalities and utilized JavaFX for graphical elements and MySQL JDBC for database connections.

The project entailed the creation of a log-in form with advanced capabilities, including user location detection, language-based translation, and error message handling. Additionally, I implemented functionalities for managing customer records and appointments, incorporating features such as auto-generation of IDs, dynamic data collection, and robust input validation. In adherence to best practices, I extensively documented the codebase with descriptive Javadoc comments, ensuring clarity and comprehensiveness. Additionally, I created a detailed README.txt file providing essential information about the application, including setup instructions, additional report descriptions, and software version details.

This project not only demonstrated my proficiency in GUI-based application development but also showcased my ability to adhere to strict requirements, implement advanced functionalities, and maintain code quality through thorough documentation and adherence to best practices.
